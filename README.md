ci03t本身也可以作为一个遥控器使用

在官方平台有通用的空调遥控代码 

设置好以后 喊出 匹配空调 就会进入匹配模式  

这时说出你的空调品牌就能自动匹配了

能匹配大多数品牌

但是只测试过一个格力空调

因为esphome空调组件没有模板类型的

因此设置了一个esphome的遥控空调来作为前端使用 

设置的遥控组件是必须的  

方案思路为  

使用语音直接让ci03t的红外遥控空调 同时ci03t通过串口把遥控的命令同步给esphome的空调组件

也可以让esphome的空调组件通过串口发送命令让ci03t遥控空调

这两个功能是双向的  

ci03t的红外电路是必须的  esphome的可以自由设置
